EMAIL CONFIGURATION GUIDE
=========================

To enable email sending for contact forms, you need to configure SMTP settings.

OPTION 1: Environment Variables (Recommended)
----------------------------------------------
Set these environment variables before running the app:

For Windows (PowerShell):
$env:SMTP_SERVER="smtp.gmail.com"
$env:SMTP_PORT="587"
$env:SMTP_USER="your-email@gmail.com"
$env:SMTP_PASSWORD="your-app-password"
$env:RECIPIENT_EMAIL="admin@omindustriesindia.com"

For Windows (Command Prompt):
set SMTP_SERVER=smtp.gmail.com
set SMTP_PORT=587
set SMTP_USER=your-email@gmail.com
set SMTP_PASSWORD=your-app-password
set RECIPIENT_EMAIL=admin@omindustriesindia.com

For Linux/Mac:
export SMTP_SERVER="smtp.gmail.com"
export SMTP_PORT="587"
export SMTP_USER="your-email@gmail.com"
export SMTP_PASSWORD="your-app-password"
export RECIPIENT_EMAIL="admin@omindustriesindia.com"

OPTION 2: Direct Configuration in app.py
-----------------------------------------
Uncomment and fill the configuration section in the send_contact_email() function
in app.py (around line 190-195).

GMAIL SETUP:
-----------
1. Enable 2-Step Verification on your Google account
2. Generate an App Password:
   - Go to: https://myaccount.google.com/apppasswords
   - Select "Mail" and "Other (Custom name)"
   - Enter "Om Industries Contact Form"
   - Copy the 16-character password
   - Use this as SMTP_PASSWORD (not your regular Gmail password)

OTHER EMAIL PROVIDERS:
---------------------
- Outlook/Hotmail: smtp-mail.outlook.com, port 587
- Yahoo: smtp.mail.yahoo.com, port 587
- Custom SMTP: Check with your email provider

TESTING:
--------
After configuration, submit a contact form and check:
1. Console output for email status messages
2. Your recipient email inbox
3. Spam folder if not received

NOTE: The contact form will still save to database even if email fails.
